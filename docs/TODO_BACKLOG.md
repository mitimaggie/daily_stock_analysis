## 待办优化项

### 添加持仓时自动拉取基础数据
- **问题**：用户添加持仓后如果从未执行过分析，stock_daily 表没有该股票的日线数据，导致 ATR 无法计算、止损为 0
- **当前案例**：512890 华泰柏瑞中证红利低波ETF，持仓 59000 股但止损为 0
- **修复方向**：在 add_portfolio() 函数中，添加持仓后自动调用数据获取模块拉取至少 60 天日线数据并写入 stock_daily 表，确保 ATR 等指标可以立即计算
- **优先级**：P1（影响资金安全——没有止损保护的持仓等于裸奔）
- **涉及文件**：src/services/portfolio_service.py (add_portfolio)、data_provider/
